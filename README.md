# Afterdeath - Bug Reports & Feature Requests

This repository is the public issue tracker for the **Afterdeath** Minecraft mod.

## How to Report a Bug
Click **[New Issue](../../issues/new)** and include:
- What happened
- What you expected to happen
- Your Minecraft version and mod version
- Steps to reproduce

  ## How to Request a Feature
  Click **[New Issue](../../issues/new)** and describe:
  - What feature you'd like
  - Why it would improve the mod
